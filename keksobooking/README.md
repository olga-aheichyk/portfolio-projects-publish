В ходе обучения на примере личного учебного проекта с готовой вёрсткой под контролем опытного наставника отработаны следующие навыки:

1. Работа с DOM
2. Работа с фунциями, методами массивов и объектов в JavaScript
3. Работа с EcmaScript-модулями
4. Описание кода с помощью JS-doc
5. Работа с API интерактивных карт библиотеки Leaflet
6. Валидация форм с помощью HTML и JavaScript
7. Настройка работы фильтров с помощью JavaScript
8. Работа с fetch, promise & callbacks в JavaScript (настройка отправки и получение данных, вывод полученных с сервера данных на страницу)
9. Обработка ошибок с помощью конструкции try - catch
10. Оптимизация работы с данными и событиями в проекте(функция debounce библиотеки lodash)
11. Настройка сборки JavaScript кода с помощью Webpack
12. Работа с Git, npm в Git Bash
